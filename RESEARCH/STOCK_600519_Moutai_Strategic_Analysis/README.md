## 关键研究问题

本研究聚焦以下7大战略领域:

### 1. 提价策略
- 飞天茅台出厂价调整时间线
- 提价幅度和频率
- 提价对收入和利润的影响
- 市场反应和消费者接受度

### 2. 渠道改革
- 传统经销商体系调整
- 直销占比变化趋势
- i茅台平台上线及运营
- 渠道利润再分配

### 3. 产能扩张
- "十三五"vs"十四五"扩产计划
- 产能利用率分析
- 技改投资情况
- 系列酒产能建设

### 4. 数字化转型
- i茅台APP用户数据和交易额
- 数字化营销体系
- 供应链数字化
- 数据中台建设

### 5. 产品创新
- 新品推出策略
- 年轻化产品线
- 系列酒品牌矩阵
- 产品结构优化

### 6. 国际化战略
- 海外市场布局
- 出口数据变化
- 国际品牌推广
- 文化输出战略

### 7. ESG与可持续发展
- 环保投入和成果
- 碳中和目标
- 社会责任项目
- 公司治理改善

## 研究方法论

### 数据收集优先级
- **A级来源**: 公司年报、公告、业绩说明会纪要
- **B级来源**: 券商研究报告、管理层公开访谈
- **C级来源**: 财经媒体报道、行业分析
- **D级来源**: 社交媒体、论坛讨论 (需交叉验证)

### 分析框架
1. **时间线分析**: 按年度梳理重大战略决策
2. **定量分析**: 收入、利润、占比等关键指标变化
3. **定性分析**: 战略意图、市场反应、竞争格局
4. **效果评估**: ROI、成功/失败判断、风险识别
