## 🎓 投资者教育

### 什么是价值投资？

价值投资是一种以**低于内在价值的价格买入优质公司**的投资方法，核心理念包括：
1. **安全边际**: 价格低于内在价值，提供下行保护
2. **护城河**: 深厚的竞争壁垒，保护长期收益
3. **长期持有**: 等待市场修正估值错误（可能需要3-5年）
4. **逆向思维**: 在市场恐慌时买入，在市场狂热时卖出

### 茅台的价值投资逻辑

**优势**:
- 护城河极深（产地、品牌、工艺、供需）
- 财务质量优异（高ROE、高利润率、充沛现金流）
- 定价权强大（历史提价100%成功）
- 长期增长确定性高

**风险**:
- 估值不便宜（P/E 30x+）
- 增长放缓（从18%降至10-12%）
- 长期挑战（人口结构、消费习惯）
- 政策风险（消费税改革、价格管控）

**投资建议**（不构成投资意见）:
- 保守投资者：等待估值回落至P/E 25x以下
- 平衡投资者：当前价位可分批建仓，定投方式
- 激进投资者：如认同长期价值，可适当配置
