## 一、数据质量评级标准

### 1.1 来源分级

| 等级 | 说明 | 示例 | 可信度 |
|------|------|------|--------|
| **A级** | 官方权威来源 | 年报、政府文件、公告 | ⭐⭐⭐⭐⭐ |
| **B级** | 专业研究 | 券商研报、行业协会 | ⭐⭐⭐⭐ |
| **C级** | 财经媒体 | 财新、一财、证券报 | ⭐⭐⭐ |
| **D级** | 其他来源 | 博客、论坛、自媒体 | ⭐⭐ (需交叉验证) |

### 1.2 验证原则

**关键数据**:
- 必须使用A级或B级来源
- 至少2个独立来源交叉验证
- 标注具体出处和日期

**敏感数据**:
- 政策解读: 优先A级来源
- 财务数据: 必须来自年报
- 客户结构: 多源验证,标注不确定性

## 八、交叉验证方法

### 8.1 财务数据验证

**方法**:
1. 年报原始数据 (A级)
2. 对比至少3家券商研报 (B级)
3. 对比财经媒体报道 (C级)
4. 数据一致则可信

**示例**:
```
2023年收入:
- 年报: 1506亿元 (A)
- 中金研报: 1506亿元 (B)
- 财新报道: 1506亿元 (C)
→ 结论: 数据可信
```

### 8.2 政策解读验证

**方法**:
1. 政府官方文件 (A)
2. 行业协会解读 (B)
3. 券商政策分析 (B)
4. 多源对比,提取共识

**示例**:
```
消费税改革:
- 财政部: 征收环节后移 (A)
- 中金: 白酒行业或纳入试点 (B)
- 招商: 2025年可能实施 (B)
→ 结论: 改革方向明确,时间存疑
```

### 8.3 市场数据验证

**方法**:
1. 多个独立来源
2. 数据范围对比
3. 时间序列对比
4. 标注不确定性

**示例**:
```
飞天茅台终端价格 (2023年):
- 来源1: ~2700元
- 来源2: ~2650-2750元
- 来源3: ~2600-2800元
→ 结论: 约2700元 (存在波动)
```

## 九、不确定性标注

### 9.1 标注原则

**必须标注不确定性的情况**:
- 客户结构数据 (公司未正式披露)
- 市场价格数据 (存在波动)
- 政策实施时间 (未公布)
- 政策影响测算 (基于假设)

### 9.2 标注格式

**示例**:
- "客户结构: 政务消费<10% (估算,需验证)"
- "i茅台用户: ~5000万 (官方未披露,待确认)"
- "消费税改革: 或于2025年实施 (讨论中,时间未定)"
